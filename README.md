# Prompt Quizzer – JavaScript Quiz Game

This project is created for *Web Dev I – Lab Assignment 4*.

Prompt Quizzer is a simple JavaScript quiz game that runs in the browser using
prompt() and alert(). The quiz asks questions one by one, checks the user’s
answers, gives instant feedback, and shows the final score at the end.

---

## Objective
To create a basic JavaScript quiz game that runs in the browser console using
arrays, loops, functions, and conditionals, without using any HTML or DOM
manipulation.

---

## Features
- Quiz questions stored in an array of objects
- Uses prompt() to take user input
- Uses alert() to show feedback and final score
- Input is validated using toLowerCase() and trim()
- Score is tracked throughout the quiz
- Runs completely in the browser console

---

## Technologies Used
- JavaScript
- HTML (only to load the JavaScript file)
